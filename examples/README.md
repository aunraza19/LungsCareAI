# Sample Medical Files

This folder contains example medical files for testing LungsCareAI.

## Audio Sample
- **H005_R4.wav** - Lung audio recording for testing audio analysis

## X-ray Samples
- **covid00186.jpg** - COVID-19 chest X-ray
- **fib.jpeg** - Pulmonary fibrosis X-ray
- **pn1.jpeg** - Pneumonia X-ray
- **xray.jpeg** - General chest X-ray

## Usage

Use these files to test the application:
1. Register a test patient
2. Upload these files in Audio Analysis or X-ray Analysis pages
3. View results and generated reports

**Note:** These are sample files for demonstration purposes only.
